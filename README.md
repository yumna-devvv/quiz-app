# Console Quiz App

A Java console-based quiz application.

## Features
- Create sections and subsections
- Add questions with multiple choices
- Delete sections, subsections, or questions
- Generate AI powered mcq (Using OpenAI)
- Take quizzes from saved questions
- Data is saved to a file

## Technologies
- Java
- File I/O
- Object-Oriented Programming
- OpenAI API (for AI generated questions)
- JSON parsing (processing API responses)
- HTTP requests (HttpURLConnection) - communicate with openAI API

## How to Run
1. Open the project in an IDE (VS Code / IntelliJ)
2. Compile the Java files
3. Run the main class
